# ruby on rails と angular で Todo

# backend
作成ようにインストールしておく

```
sudo apt install ruby
sudo gem install rails

or

sudo apt install ruby-railties
```

作成する
```
rails new backend -d mysql --skip-bundle
```

# frontend

## 作成
作成ようにインストールしておく

```
npm install -g @angular/cli
```

作成する
```
ng new frontend
```
